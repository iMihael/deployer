Add Symlink

Add symlink for shared files in project, for example to images or config files that will be the same from one release to other.

Source file or folder path is required.
Destination symlink path is required.
You can type {RELEASE_FOLDER} to use latest release directory path, for example {RELEASE_FOLDER}/web/photos
